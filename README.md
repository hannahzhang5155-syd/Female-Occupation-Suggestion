# 职属于她 Demo 使用说明

这是 FemAI 团队「职属于她」网页 Demo，核心流程为：

1. 用户进入首页
2. 完成第一套职业人格选择题
3. 系统根据五类人格权重计算主导人格
4. 用户继续完成第二套现实条件选择题
5. Demo 从 40 个职业赛道中，根据人格分数、行业兴趣、技能、工作方式、城市、目标、时间、预算等信息推荐 3 个赛道
6. 结果页生成 6 个月行动路线图和可复制的深度分析 Prompt

## 本地打开方式

直接双击 `index.html` 即可预览。

如果浏览器拦截本地文件，可以在当前文件夹启动本地预览服务：

```bash
python3 -m http.server 4174 --bind 127.0.0.1
```

然后打开：

```text
http://127.0.0.1:4174/
```

## 当前 Demo 策略：不直接接入 OpenAI API

为了让一周 Demo 更稳定，当前版本不要求用户提供 API Key，也不在网页里直接调用 OpenAI。

产品逻辑是：

1. 职属于她先根据问卷给出复合职业人格、3 个推荐赛道和 6 个月路线图
2. 如果用户想继续深入，系统自动生成一段高质量 Prompt
3. 用户复制 Prompt，粘贴到自己的 AI 工具里继续探讨

这样做的好处：

- 不需要处理 OpenAI API Key
- 不产生 API 调用成本
- 不会因为额度、网络或账号问题影响展示
- 用户可以使用自己习惯的 AI 工具，例如 ChatGPT、Claude、豆包、Kimi 等

## 第一套题：人格判断

已使用你提供的 V4 版 32 道题，判断五类职业人格：

- 创业型人格
- 技能型人格
- 社交型人格
- 创意型人格
- 内容创作型人格

当前评分规则：

- 每道题的 A-E 选项分别对应创业型、技能型、社交型、创意型、内容创作型，具体以 V4 题库中的 dimension 字段为准
- 用户选择的选项会给对应人格加分
- 系统会按每类人格可获得的最高分进行标准化
- 标准化分数最高者为主导人格，第二高者为辅助人格
- 结果页会根据“主导人格 + 辅助人格”生成复合人格描述

## 第二套题：赛道推荐信息采集

第二套题不是为了再次判断人格，而是为了从 40 个职业赛道中筛出更适合用户的方向。当前包含：

- 当前状态
- 职业目标
- 行业兴趣
- 已有技能或资源
- 偏好的工作方式
- 每周可投入时间
- 学习深度
- 启动资金
- 风险承受程度
- 城市/地域条件
- 现实约束
- 希望产出的可见成果

当前赛道库已扩展到 40 个，包含 AI、数据、金融、投资、建筑、空间、产品、咨询、运营、教育、内容、品牌、设计、公益、ESG、美业和本地服务等方向。

## 未来如果要正式接入 ChatGPT

如果后续要把 AI 分析直接放进页面，建议再加入后端接口。注意：不要把 OpenAI API Key 写在前端网页里。

推荐架构：

```text
前端网页
  ↓ 发送人格分数 + 第二套问卷答案
后端接口 /api/analyze-career
  ↓ 安全调用 OpenAI API
OpenAI 返回结构化分析
  ↓
前端渲染结果页
```

但当前一周 Demo 不走这条链路。当前版本只生成 Prompt，让用户复制到自己的 AI 工具里继续分析。

## 一周 Demo 建议

第一版不需要追求复杂平台，重点是让用户完成测评后觉得：

> “这个建议有点懂我，而且我知道下一步可以做什么。”

所以展示时建议强调：

- 不是给用户下定论，而是帮助她缩小探索范围
- 不是承诺收入，而是给出低成本验证路径
- 不是单纯测人格，而是结合现实条件做推荐

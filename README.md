# 女性职业转型探测器 Demo 使用说明

这是 FemAI 团队第一版「女性职业转型探测器」网页 Demo，核心流程为：

1. 用户进入首页
2. 完成第一套职业人格选择题
3. 系统根据五类人格权重计算主导人格
4. 用户继续完成第二套现实条件选择题
5. Demo 根据人格分数、兴趣、技能、城市、目标、时间、预算等信息推荐 3 个赛道
6. 结果页生成 6 个月行动路线图和可复制的深度分析 Prompt

## 本地打开方式

直接双击 `index.html` 即可预览。

如果浏览器拦截本地文件，可以在当前文件夹启动本地预览服务：

```bash
python3 -m http.server 4174 --bind 127.0.0.1
```

然后打开：

```text
http://127.0.0.1:4174/
```

## 当前 Demo 策略：不直接接入 OpenAI API

为了让一周 Demo 更稳定，当前版本不要求用户提供 API Key，也不在网页里直接调用 OpenAI。

产品逻辑是：

1. 女性职业转型探测器先根据问卷给出基础职业人格、3 个赛道和 6 个月路线图
2. 如果用户想继续深入，系统自动生成一段高质量 Prompt
3. 用户复制 Prompt，粘贴到自己的 AI 工具里继续探讨

这样做的好处：

- 不需要处理 OpenAI API Key
- 不产生 API 调用成本
- 不会因为额度、网络或账号问题影响展示
- 用户可以使用自己习惯的 AI 工具，例如 ChatGPT、Claude、豆包、Kimi 等

## 第一套题：人格判断

已使用你提供的 25 道题，判断五类职业人格：

- 创业型人格
- 技能型人格
- 社交型人格
- 创意型人格
- 内容创作型人格

当前评分规则：

- 李克特量表题：1-5 分累加到对应人格
- 情境/偏好选择题：用户选择的选项给对应人格加 5 分
- 最终得分最高者为主导人格，第二高者为辅助人格

## 第二套题：赛道推荐信息采集

第二套题不是为了再次判断人格，而是为了让推荐更落地。当前包含：

- 年龄范围
- 当前状态
- 当前月收入范围
- 所在/目标城市范围
- 兴趣方向
- 已有技能或资源
- 每周可投入时间
- 启动资金
- 职业目标
- 风险承受程度

建议第二版可新增：

- 当前行业与岗位
- 最高学历 / 专业背景
- 是否需要照护家庭、时间是否碎片化
- 是否愿意露脸 / 公开表达
- 是否有现成人脉或私域资源
- 是否有作品集、案例或过往项目
- 目标收入时间线：3 个月、6 个月、12 个月
- 更偏好线上、线下还是混合工作
- 最不能接受的工作状态，例如高压销售、频繁出差、长期加班
- 用户最担心的问题，用 1 道开放题收集

## 未来如果要正式接入 ChatGPT

如果后续要把 AI 分析直接放进页面，建议再加入后端接口。注意：不要把 OpenAI API Key 写在前端网页里。

推荐架构：

```text
前端网页
  ↓ 发送人格分数 + 第二套问卷答案
后端接口 /api/analyze-career
  ↓ 安全调用 OpenAI API
OpenAI 返回结构化分析
  ↓
前端渲染结果页
```

但当前一周 Demo 不走这条链路。当前版本只生成 Prompt，让用户复制到自己的 AI 工具里继续分析。

## 一周 Demo 建议

第一版不需要追求复杂平台，重点是让用户完成测评后觉得：

> “这个建议有点懂我，而且我知道下一步可以做什么。”

所以展示时建议强调：

- 不是给用户下定论，而是帮助她缩小探索范围
- 不是承诺收入，而是给出低成本验证路径
- 不是单纯测人格，而是结合现实条件做推荐
